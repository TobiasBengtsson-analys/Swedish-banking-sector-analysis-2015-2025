# Swedish Banking Sector Analysis (2015–2025)

This project analyses the balance sheet development of the three major Swedish banks:
SEB, Swedbank, and Handelsbanken using SCB financial market statistics.

## Objective
To describe structural changes in bank balance sheets with focus on:
- Total assets
- Lending development
- Funding structure
- Loan-to-deposit ratios (system-level financing gap)

## Data
Data is sourced from Statistics Sweden (SCB) financial market statistics for monetary financial institutions (MFI).

## Methodology
- Data cleaning and reshaping using pandas
- Construction of balance sheet indicators
- Calculation of structural ratios (LTA, funding structure, loan-to-deposit proxy)
- Monthly time series analysis (2015–2025)

## Key Outputs
- Balance sheet development over time
- Funding structure composition
- Structural dependence on capital markets

## Figures
All figures are stored in `/figures`.

## Limitations
The analysis is purely descriptive and does not assess financial stability.
